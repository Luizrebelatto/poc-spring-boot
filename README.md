# Api Spring Boot

## How to run
1. Open intellij
2. Run api and access `http://localhost:8080`
3. Run docker `docker-compose up`

# Routes
- GET `http://localhost:8080/api/hello`
- Actuator `http://localhost:8080/actutuator`
- Prometheus `http://localhost:9090`
- Grafana `http://localhost:3000`
